### Client Server
Вариант задания 2: страница, отображающая в браузере древовидную структуру.
Использован Spring Framework
Приложение отображает иерархию каталогов компьютера, начальный каталог задаётся в файле ClientServer/src/main/resources/path.properties
Для изменения имени каталога необходимо нажать на имя каталога, затем нажать F2
